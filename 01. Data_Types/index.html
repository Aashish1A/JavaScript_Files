Data Types in JavaScript

There are two types of data types in JavaScript

  1. Primitive (values type)
  2. Non Primitive (Reference type)

We can classify Primitive data types in 7 categories.

  1. string
  2. number
  3. boolean
  4. undefined
  5. null
  6. bigint
  7. symbol

There are only one type of non primitive data type that is called object.

  • Object Literals
  • Arrays
  • Functions
