What are Promises in JavaScript?

A Promise in JavaScript represents a value that may be available now, later, or never. It's used to handle asynchronous operations like API calls, file reading, timers, etc. Promises help avoid callback hell, making the code cleaner and easier to follow.

States of a Promise
    A Promise has three states:
  1. Pending: The initial state, neither fulfilled nor rejected.
  2. Fulfilled: The operation was completed successfully.
  3. Rejected: The operation failed.
Structure of a Promise
    A Promise is created using the new Promise() constructor and takes a function with two parameters:
  1. resolve: Call this when the operation completes successfully.
  2. reject: Call this when the operation fails.
